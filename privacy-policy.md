Your privacy is important. This extension is designed to work fully on your device without collecting or selling any personal data.

---

## Data Collection
**This extension does NOT collect, transmit, or sell any personal data.**

All features work locally inside your browser.

---

## Clock & Time Information
- The clock uses your system time and browser timezone
- No clock data is stored or shared
- Time and timezone are accessed locally using browser APIs
- Clock data never leaves your device

This feature is fully secure and offline.

---

## Weather Feature
- Weather is **optional**
- Location access is requested **only after user clicks the weather box**
- Location is used temporarily to fetch weather data
- Location data is **never stored**
- Location is **never sent to any third party other than the weather API**

Weather data is fetched from:
- Open-Meteo (no API key, no tracking)

---

## Search & Recent Activity
- Search queries are stored locally to show recent searches
- Recent searches are saved only in your browser
- No search data is transmitted or shared

You can clear this data anytime by clearing extension storage.

---

## To-Do List
- Tasks are saved locally using browser storage
- No task data is shared or uploaded
- Tasks never leave your device

---

## Background Media (Images & Videos)
- Uploaded backgrounds are stored locally
- Media files are saved inside your browser storage
- No uploads to any server
- No background data is shared

---

## Quick Links / Bookmarks
- Links are saved locally in your browser
- No browsing history is tracked
- Clicking a link opens the official website in a new tab

---

## Permissions Explanation
The extension may request the following permissions:

- **Geolocation**  
  Used only for optional weather functionality  
  Requested only after user interaction  

- **Storage**  
  Used to save preferences, tasks, links, and backgrounds locally  

No permission is used for tracking or advertising.

---

## Third-Party Services
This extension may access:
- Open-Meteo (weather data)
- Google favicon service (website icons)

These services are used only to improve user experience.

---

## Data Security
- No external databases are used
- No analytics or trackers
- No ads
- No accounts required

All data remains under your control.

---

## Changes to This Policy
If privacy practices change, this policy will be updated accordingly.

---

## Contact
If you have questions about privacy, you can contact the developer via the Chrome Web Store listing.
